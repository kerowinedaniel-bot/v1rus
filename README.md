# V1RUS HUB — Ultra-modern site (Flask)

Local development:

1. Create a virtual environment and install dependencies:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

2. Run the site:

```bash
python app.py
# or
flask run --port 5000
```

Open http://127.0.0.1:5000

The site uses HTML/CSS/JS with a canvas-based particle background, glassmorphism, neon effects and an animated developer terminal.
